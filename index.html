Selected Projects

Secure Encrypted Chat

I built a Python-based desktop application using Tkinter to make cryptography practical. It allows users to securely encrypt and decrypt text or files using AES, DES, and 3DES. The tool focuses on real-world security by using PBKDF2 and a salt to turn passphrases into strong cryptographic keys. It also includes a real-time performance logger and graph to compare the speed of each algorithm.

Why I Did It: I built this project because I was tired of just reading about cryptography. I wanted to jump into the deep end and see how AES, DES, and key derivation actually work in a real program. My core motivation was to gain a hands-on understanding of secure passphrase management using hashing like SHA-256 and how these algorithms behave in a real-world system.

What I Learned:

  • Developed a strong understanding of secure passphrase handling using PBKDF2 and SHA-256, and the distinction between a passphrase and a cryptographic key.
  • Measured and compared encryption and decryption performance using high-resolution timing, confirming AES as the fastest and observing unexpected real-world differences between DES and 3DES.
  • Gained hands-on experience integrating system-level tools like OpenSSL into a Python application for broader compatibility.
  • Connected GUI design, user input validation, encryption logic, file handling, and performance analysis into a single cohesive application.
