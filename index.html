Project Spotlight: Secure Encrypted Chat Dashboard

I built a Python-based desktop application using Tkinter to make cryptography practical. It allows users to securely encrypt and decrypt text or files using AES, DES, and 3DES. Why I Did It: My goal was to move beyond theory and tackle the practical challenges of encryption, including secure key handling, algorithm performance, and system-level integration using OpenSSL.

What I Learned:

  • Secure passphrase handling with PBKDF2 and SHA-256
  • Real-world performance differences between AES, DES, and 3DES
  • System-level cryptography integration using OpenSSL
