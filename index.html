Selected Projects

Secure Encrypted Chat Dashboard

A Python-based desktop application built with Tkinter that enables secure encryption and decryption of text and files using AES, DES, and 3DES. The application uses PBKDF2 with salting to derive cryptographic keys from user passphrases and includes real-time performance logging to compare algorithm speed.

Why I Did It: To move beyond cryptography theory and understand how encryption behaves in real-world applications, including key management, performance tradeoffs, and system-level integration challenges.

What I Learned:

  • Secure passphrase handling using PBKDF2 and SHA-256
  • Real-world performance differences between AES, DES, and 3DES
  • System-level cryptography integration using OpenSSL
  • End-to-end application design combining GUI and backend logic
