Selected Projects

Project Spotlight: Secure Encrypted Chat Dashboard

I built a Python-based desktop application using Tkinter to make cryptography practical. It allows users to securely encrypt and decrypt text or files using three major symmetric algorithms: AES, DES, and 3DES. The tool focuses on real-world security by using PBKDF2 and a salt to turn passphrases into strong cryptographic keys. It also features a real-time performance logger and graph to compare the speed of the three algorithms.

Why I Did It: My goal was to move beyond theory and tackle the practical challenges of encryption. I wanted to see how algorithms truly behave, how to securely manage user passphrases using SHA-256 hashing and key derivation, and what engineering issues arise. When I ran into compatibility problems with Python's native crypto libraries, I pivoted to integrating OpenSSL via subprocess calls.

What I Learned:

  • Secure passphrase handling with PBKDF2 and SHA-256
  • Performance differences between AES, DES, and 3DES
  • System-level cryptography integration using OpenSSL
  • End-to-end desktop application design
